# بناء تطبيق حلقتي بدون Android Studio

هذه الحزمة تحتوي على مشروع Android جاهز، ومعه ملف GitHub Actions يبني ملف APK تلقائيًا عبر السحابة.

## الطريقة من الهاتف أو الكمبيوتر

1. أنشئ حسابًا في GitHub إن لم يكن لديك.
2. من GitHub اضغط New repository.
3. سمّه مثلًا: `HalaqatiAndroid`.
4. ارفع ملفات هذا المجلد كاملة إلى المستودع.
   - مهم: ارفع مجلدات `app` و `.github` وملفات `build.gradle` و `settings.gradle`.
5. افتح تبويب Actions داخل المستودع.
6. اختر workflow باسم: `Build Android APK`.
7. اضغط: `Run workflow`.
8. بعد انتهاء البناء افتح آخر تشغيل ناجح.
9. من أسفل الصفحة حمّل الملف من Artifacts باسم: `Halaqati-debug-apk`.
10. فك ضغطه وستجد ملف APK.
11. انقل APK للهاتف وثبّته. قد تحتاج إلى تفعيل: السماح بالتثبيت من مصادر غير معروفة.

## ملاحظات

- هذا APK تجريبي Debug، مناسب للتثبيت الشخصي والتجربة.
- للنشر على متجر Google Play تحتاج إصدار Release موقّع بمفتاح خاص.
- التطبيق يستخدم WebView ويشغل ملف `app/src/main/assets/index.html` داخليًا.
- إذا أردت تحديث التطبيق لاحقًا، استبدل ملف `index.html` داخل assets ثم شغّل workflow مرة أخرى.

## حل مشكلة عدم ظهور Actions

إذا لم يظهر زر Run workflow، نفّذ تعديلًا بسيطًا على أي ملف أو ارفع الملفات مرة أخرى، ثم افتح Actions ووافق على تشغيل workflows إن طلب GitHub ذلك.
